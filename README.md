




# MGSANet
Tensorflow implementation of MGSANet: Multi-Granularity Scale-Aware Networks for Hard Pixels Segmentation of Pulmonary Nodules



# Requirements
Please, install the following packages
* numpy
* tensorflow-gpu>=1.14
* cv2
* h5py
* tqdm
